### 크롬 캡쳐

```
크롬 개발자모드(F12) > Toggle device toolbar (Ctrl +Shift + M) >
Dimensions: Responsive 설정 (1000 x 600) > ...버튼 > Capture full size screenshot, Capture screenshot
```

### 크롬 네트워크 설정

```
크롬 개발자모드(F12) > Network tab > Fast 3G, Slow 3G 등 선택
```
